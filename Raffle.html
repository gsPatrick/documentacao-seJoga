POST /raffles/:raffleId/buy-tickets

Compra bilhetes específicos para uma rifa.

Parâmetros:

  • `raffleId` (number, path): ID da rifa.

Headers

  • `Authorization: Bearer [seu_token_jwt_valido]`
  • `Content-Type: application/json`

Request Body:

{ "ticketNumbers": ["01", "05", "12"], "type": "tradicional" // ou "equipes" }

Retorno

201 Bilhetes comprados com sucesso.

[ { "id": 1, "userId": 1, "raffleId": 1, "ticketNumber": "01", "createdAt": "2025-01-25T16:25:12.345Z", "updatedAt": "2025-01-25T16:25:12.345Z" }, { "id": 2, "userId": 1, "raffleId": 1, "ticketNumber": "05", "createdAt": "2025-01-25T16:25:12.345Z", "updatedAt": "2025-01-25T16:25:12.345Z" }, { "id": 3, "userId": 1, "raffleId": 1, "ticketNumber": "12", "createdAt": "2025-01-25T16:25:12.345Z", "updatedAt": "2025-01-25T16:25:12.345Z" } ]

Parâmetros:

  • `raffleId` (number, path): ID da rifa.

Headers

  • `Authorization: Bearer [seu_token_jwt_valido]`
  • `Content-Type: application/json`

Request Body:

{ "quantity": 3, "type": "tradicional" }
